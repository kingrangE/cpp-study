# MyStringTest

씹어먹는 C++ 실습 중 문자열 클래스 만들기입니다. ([코드](https://github.com/kingrangE/cpp-study/tree/main/MyStringTest))
저는 Python String처럼 조작할 수 있도록 몇 가지 함수를 더 추가했습니다.
- 총 기능
    1. 문자열 출력 함수
    2. 문자열 길이 반환 함수
    3. 문자열 뒤에 다른 문자열 붙이기
    4. 문자열 내에 포함되어 있는 문자열 위치 반환
    5. 문자열 같은지 비교
    6. 문자열 크기 비교
    7. 재할당
    8. insert 함수
    9. join
    10. at
    11. erase
    12. replace
